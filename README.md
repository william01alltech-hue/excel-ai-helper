# Excel AI 智慧助手 - 本地開源版 Sideload 指南

本專案是一個本地運行的 Excel AI 側邊欄增益集，能夠與您電腦本機安裝的 **Ollama** 串接，協助分析 Excel 格式並自動生成假單或工時表。

---

## 🚀 快速啟動步驟

### 步驟 1：安裝 Node.js 依賴套件
在專案目錄下執行以下指令以安裝開發伺服器所需的套件（包括本地 HTTPS 支援）：
```bash
npm install
```

### 步驟 2：啟動 Ollama 並允許跨網域存取 (CORS)
由於 Excel 增益集是在網頁沙盒環境執行，您必須啟動 Ollama 並**允許跨網域存取**。

請開啟**終端機**，輸入以下指令啟動 Ollama 服務：
```bash
OLLAMA_ORIGINS="*" ollama serve
```
*(如果終端機顯示連接埠已被佔用，請先在 Mac 工作列將 Ollama 圖示點右鍵結束 (Quit)，再重新執行上述指令。)*

接著在另一個終端機視窗，下載您想使用的繁體中文開源模型（推薦 `qwen2.5` 或 `gemma2`）：
```bash
ollama run qwen2.5
```

### 步驟 3：啟動本地增益集網頁伺服器
在專案目錄下執行：
```bash
npm run dev
```
這會啟動一個本地的安全 HTTPS 伺服器，網址為 `https://localhost:5173/`。

> 💡 **第一次啟動的重要步驟**：
> 請在瀏覽器中手動打開 `https://localhost:5173/`，瀏覽器可能會顯示「您的連線並非安全連線」。
> 請點擊**「進階」** -> **「繼續前往 localhost (不安全)」**。這是為了讓您的 Mac 信任本地開發憑證，否則 Excel 會阻擋載入網頁。

### 步驟 4：在 Mac Excel 中載入 (Sideload) 增益集
1. 打開 Finder，使用快速鍵 `Cmd + Shift + G` 打開「前往資料夾」視窗。
2. 輸入以下路徑並前往：
   ```
   ~/Library/Containers/com.microsoft.Excel/Data/Documents/
   ```
3. 在該目錄下建立一個名為 `wef` 的新資料夾（如果已經存在則忽略）。
4. 將本專案中的 `manifest.xml` 複製並貼上到 `wef` 資料夾中。
5. **重啟或開啟 Excel**。
6. 新建或打開一個活頁簿，點擊上方選單的 **「插入」 (Insert) -> 「我的增益集」 (My Add-ins) 下拉選單**，您會看到 **「Excel AI 智慧助手」**。
7. 點擊它，右側即會出現精美的 AI 側邊欄！

## 🛠️ 使用說明
1. **連線測試**：進入側邊欄後，點擊「Ollama 本地連線設定」展開，確保狀態顯示為「已連線」，並在下拉選單選取您下載的模型（例如 `qwen2.5`）。
2. **分析格式與生成**：
   * 在 Excel 中，用滑鼠選取您的「假單標頭」或「工時表標頭」（例如選取姓名、假別、起始時間、結束時間等欄位）。
   * 點擊側邊欄下方的 **「📋 分析選取格式」**，AI 會自動讀取這些欄位並在對話框中生成模擬資料。
   * 點擊下方的 **「✍️ 將 AI 生成填入 Excel」**，生成的資料就會整齊地填回您的儲存格中！
